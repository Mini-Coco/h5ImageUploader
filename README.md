# h5ImageUploader
一个具有图片旋转、压缩、下载功能的h5上传组件

# 功能说明
- 项目基于vue
- 可用于pc端、h5
- 当上传图片的时候会根据limitsize属性对图片进行压缩
- ios拍照会出现图片旋转的情况，该组件对这种旋转情况进行了修复: https://blog.csdn.net/xiaoermingn/article/details/94398621
- 可对已上传的图片进行下载
